# 99 problems

## Historical note

Originally, it is a list of problems for practicing logic programming in Prolog assembled by professor Werner Hett.

## List of problems with solutions in different languages

- [Original P-99 (Prolog)](https://sites.google.com/site/prologsite/prolog-problems)
- [L-99 (LISP)](http://www.ic.unicamp.br/~meidanis/courses/mc336/2006s2/funcional/L-99_Ninety-Nine_Lisp_Problems.html)
- [H-99 (Haskell)](https://wiki.haskell.org/H-99:_Ninety-Nine_Haskell_Problems)
- [O-99 (OCaml)](https://ocaml.org/learn/tutorials/99problems.html)
- [S-99 (Scala)](http://aperiodic.net/phil/scala/s-99/)
